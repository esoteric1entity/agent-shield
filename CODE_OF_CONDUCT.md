# Code of Conduct

## Our standard

This project is a place to make AI agents safer and more reliable. Everyone
participating — maintainers, contributors, reporters — is expected to:

- **Be respectful.** Critique code and ideas, never people.
- **Be constructive.** "This breaks X, here's a repro" beats "this is bad."
- **Assume good faith.** Most disagreements are unshared context.
- **Stay honest.** Claims about behavior come with evidence; security claims
  doubly so. This is a security project — overstating protection harms users.

Unacceptable behavior includes personal attacks, harassment of any kind,
publishing others' private information, and sustained disruption of
discussions.

## Scope

Applies in all project spaces — issues, pull requests, discussions, security
reports — and when representing the project elsewhere.

## Enforcement

Report conduct issues privately via GitHub (open a private report through the
repository's Security tab, marked "conduct", or contact the maintainer
through GitHub). Maintainers will review confidentially and respond
proportionally: a reminder, a warning, or removal from project spaces for
repeated or severe violations. Maintainers who violate this code are subject
to the same standard.
